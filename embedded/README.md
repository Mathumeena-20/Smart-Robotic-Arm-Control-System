# 🤖 Smart Robotic Arm – Fullstack Project

A complete fullstack robotics platform integrating:

- 🧠 **Backend**: Flask (Python)
- 🎛️ **Frontend**: ReactJS
- 🔌 **Embedded Simulation**: Python
- 📡 **Messaging**: MQTT (Mosquitto)
- 🗄️ **Database**: PostgreSQL
- 🐳 **Dockerized Services**
